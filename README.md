# ระบบเก็บค่าหอพักนักศึกษา-Repository-Structure

ระบบบริหารจัดการยอดค้างชำระและเก็บค่าหอพักนักศึกษาดิจิทัล
(Student Dormitory Payment and Billing Management System)
📋 ภาพรวมโครงการ (Project Overview)
แพลตฟอร์มดิจิทัลที่ออกแบบมาเพื่อเพิ่มประสิทธิภาพในการบริหารจัดการค่าใช้จ่ายภายในหอพักนักศึกษา ช่วยให้นิสิตสามารถตรวจสอบยอดค้างชำระรายเดือน เช่น ค่าเช่าห้อง ค่าน้ำ และค่าไฟ ได้ด้วยตนเองผ่านระบบออนไลน์ โดยนิสิตสามารถแจ้งชำระเงินผ่านการอัปโหลดหลักฐานการโอนเงิน (Slip) และติดตามสถานะการตรวจสอบจากเจ้าหน้าที่ได้แบบเรียลไทม์ เพื่อลดขั้นตอนการทำงานที่ซับซ้อน ลดการใช้กระดาษ และเพิ่มความโปร่งใสในการจัดเก็บรายได้

🚀 ฟีเจอร์หลักของระบบ (Key Features)
Dashboard สำหรับนิสิต: ตรวจสอบยอดค้างชำระแยกตามประเภท (ค่าเช่า/น้ำ/ไฟ)

ระบบแจ้งชำระเงิน: อัปโหลดสลิปหลักฐานการโอนเงินและระบุวันที่เวลาที่โอน

ระบบแจ้งเตือน: แจ้งเตือนเมื่อบิลใหม่ถูกสร้าง หรือเมื่อเจ้าหน้าที่ยืนยันยอดเงินสำเร็จ

ระบบจัดการสำหรับเจ้าหน้าที่: ออกใบแจ้งหนี้รายเดือน ตรวจสอบสลิป และสรุปรายงานรายรับ

Responsive Design: รองรับการใช้งานอย่างเต็มรูปแบบทั้งบนคอมพิวเตอร์และสมาร์ทโฟน
